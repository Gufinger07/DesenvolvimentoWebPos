Exemplo 11 CSS - Box Model

Box Model

O modelo de baixa(Box model) DO CSS é um conceito fundamental que define como os elementos HML sâo representados como caixas retangulares no documento html. Essa representação permite o controle preciso dos layouts e espaçamentos.

componentes do Box Model

  1. Conteúdo(Content)Area onde o conteúdo, como texto, imagens, paragrafos, etc serão exibidos. Suas dimensões são definidas pelas propriedades height e width
  2. Padding(Preenchimento)Espaço invisivel entre conteudo e a borda deste conteúdo
  3. Border(Borda)Linha que se circunda o conteúdo e o preenchumento Padding, podendo ter expressura, estilo e cores definidas
  4. Morgin(margin)Espaço transparente externo à borda utilizado para separar o elemento dos outros elementos do documento HTML
