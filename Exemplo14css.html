Exemplo 14 CSS - FlexBox

O Flexbox é um modelo de layout em css que foi proejtado para facilitar a criação de layouts complexos. Ele é especialmente útil para criação de layouts responsivos.

Principais propriedades do Flexbox

Flex-Direction

A propriedade flex-direction permite alterar a direção na qual os elementos html sreão exibidos. A seguir demonstramos o seu uso.
