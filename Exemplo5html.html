Exemplo 5 html - TAGs de seção ou títulos

Este exemplo mostra como usar tags de título( <h1> a <h6> ) para organizar o conteúdo de um documento html em seções.

Descrição

As tags de título são usadas para definir cabeçalhos e subtitulos. Elas variam de code <h1> a <h6> mais importante a <h1> a <h6> menos importante

Exemplo de uso

Exemplo de titulo: usei a tag h1

Exemplo de subtitulo: usei a tag h2

Exemplo de subtitulo: usei a tag h3

Exemplo de subtitulo: usei a tag h4

Exemplo de subtitulo: usei a tag h5
Exemplo de subtitulo: usei a tag h6

Neste exemplo, mostramos como usar títulos de diferentes níveis em html. Cada título representa um nível de importância do conteúdo.
