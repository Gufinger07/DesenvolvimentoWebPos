Exemplo 6 HTML - Elementos HTML Listas Ordenadas, Não ordenadas e âncora.

Listas

Lista em html são representadas pelas tags <ol> que representam listas ordenadas e <ul> que representam listas ordernadas. Estas listas possuem diversas finalidade, podendo ser utilizadas para exibir informações ou simplesmente serem exibidas como um menu.

Toda vez que usar listas <ol>, <ul> devemos sempre utilizar a tag de item <li>

Exemplo de lista ordenada tag ol

  1. Item 1
  2. Item 2
    1. subitem2.1
    2. subitem2.2
    3. subitem2.3
    4. subitem2.4
  3. Item 3
  4. Item 4

Exemplo de lista tag ul

  • Item 1
  • Item 2
    • subitem2.1
    • subitem2.2
      • subitem 2.2.1
      • subitem 2.2.2
      • subitem 2.2.3
      • subitem 2.2.4
    • subitem2.3
    • subitem2.4
  • Item 3
  • Item 4

Elemento HTML âncora - tag <a> é usada para criar links ou ancoras em uma página web. Esse elemento html tem como função principar permitir o link entre elementos hipertexto, tais como arquivos, documentos etc...

Alêm disso, a tag <a> é muito utilizada para promover a navegação na web.
