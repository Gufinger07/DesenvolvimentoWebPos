Exemplo 4 - Estrutura Básica de um Documento HTML

Este exemplo mostra a estrutura minima de um documento HTML. A seguir são descritas as principais tags utilizadas

  • <!DOCTYPE html >:Define o tipo do documento HTML
  • <html >:É o elemento central, o container do documento, de toda a página
  • <head >:Contem os metadados que serão interpretados pelo navegador, como título das pagina e links para arquivos extrnos
  • <body >: Onde todo o conteúdo ficará visivel na página

Abaixo está o código completo deste exemplo

        <!DOCTYPE html>
        <html lang="pt-BR">
          <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Exemplo 1 - Estrutura Básica</title>
          </head>
          <body>
            <h1>Bem-vindos à minha página!</h1>
            <p>Conteúdo principal aqui</p>
            <p>Rodapé da página</p>
          </body>
        </html>
