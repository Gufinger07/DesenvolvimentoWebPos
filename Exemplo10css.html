Exemplo 7 - CSS - Vinculação Externa de Arquivo CSS

Vinculação Externa

A vinculação de arquivos CSS externa é aquela onde um arquivo CSS é vinculado a um documento html por meio do elemento link. Esse tipo de vinculação é a mais recomendada para quando temos muitas páginas HTML.

Vinculação Incorporada - Interna

A vinculação incorporada é aquela que as regras CSS são colocadas diretamente no documento HTML, dentro da tag STYLE. Esse tipo de vinculação só é recomendado quando temos uma ou duas páginas no nosso projeto.

Vinculação Inline

A vinculação inline é aquela onde vinculamos uma regra CSS diretamente em um elemento HTML. Neste caso também usamos a tag STYLE, porém ela é aplicada diretamente no elemento o que pode dificultar a sua manutenção. Logo use-a quando necessário, com muito cuidado.
