Exemplo 9 - CSS Seletores

Seletores CSS

Os seletores CSS são padrões usados para selecionar e estilizar elementos HTML em uma página web. Eles permitem que você aplique estilos específicos a elementos individuais, grupos de elementos ou elementos que atendam a certas condições. Os seletores são uma parte fundamental do CSS (Cascading Style Sheets) e são usados para definir como os elementos devem ser exibidos.

Exemplo de seletor de id

Seletor de id é aquele que seleciona um único elemento com um atributo id específico. Exemplo: #cabecalho { font-size: 24px; } aplica um tamanho de fonte de 24px ao elemento com o ID cabecalho

Seletor de classe

O seletor de classe no CSS é usado para selecionar e estilizar elementos HTML que possuem um atributo class específico. Ele é um dos seletores mais comuns e poderosos, pois permite aplicar estilos a múltiplos elementos que compartilham a mesma classe, independentemente do tipo de elemento.

Seletor de Tipo

O seletor de tipo (também conhecido como seletor de tag) no CSS é usado para selecionar e estilizar todos os elementos de um determinado tipo (ou tag) HTML. Ele é um dos seletores mais básicos e diretos, pois aplica estilos a todos os elementos que correspondem ao nome da tag especificada.
