Exemplo tabelas

Descrição

Este exemplo mostra como podemos usar a tag table para criar tabelas. Tabelas devem ser utilizadas para mostrar dados tubulares nunca para outra finalidade

  • Tag table - Esta tag define uma tabela
  • Tag tr - Esta tag define uma linha da tabela
  • Tag th - Esta tag define uma o cabeçalho da tabela
  • Tag td - Esta tag define o conteudo da tabela
