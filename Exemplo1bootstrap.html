Exemplo 1 Bootstrap - Principais elementos

Neste exemplo utilizamos o container, que é seletor de classe disponibilizado pelo Bootstrap. Esse seletor ~e utilizado quando queremos estruturar nosso documento html e alinhar seu conteúdo de forma responsiva

O seletor Container possui largura fixa que se adapta ao tamanho do viewPort. Esse container é ideal para documentos que precisam de margens laterais
